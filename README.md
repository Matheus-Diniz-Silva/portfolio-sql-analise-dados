# Portfólio de Análise de Dados com SQL

Bem-vindo ao meu repositório de projetos SQL! Este espaço demonstra minhas habilidades em SQL para Análise de Dados, desde consultas básicas até análises complexas para suporte à tomada de decisão.

## Sobre

Sou um Analista de Dados em formação e aqui documento minha jornada de aprendizado e aplicação prática do SQL. Este portfólio apresenta dois projetos completos que mostram minha evolução desde conceitos fundamentais até análises estratégicas avançadas.

## Habilidades Demonstradas

- **Consultas SQL**: SELECT, WHERE, ORDER BY, GROUP BY, HAVING
- **Junções**: INNER JOIN, LEFT JOIN, RIGHT JOIN
- **Agregações**: COUNT, SUM, AVG, MIN, MAX
- **Funções de Data**: DATE_FORMAT, TIMESTAMPDIFF, MIN/MAX dates
- **CTEs (Common Table Expressions)**
- **Funções de Janela**: ROW_NUMBER(), LAG/LEAD
- **Análise de Cohort e Retenção**
- **Análise para Negócios**: Criação de métricas e KPIs estratégicos

## Projetos

### [Projeto 01: Análise de Vendas Básica](/01-projeto-basico)
**Objetivo:** Análise inicial de dataset de vendas para identificar tendências e métricas fundamentais.

**Principais Descobertas:**
- Padrões claros de sazonalidade (Novembro: +96% crescimento)
- Eficiência em Large Deals (25% do faturamento com 5.56% das vendas)
- Ticket médio variável: Small ($2,062) | Medium ($4,398) | Large ($8,294)

**Tecnologias:** MySQL, Agregações, Análise Temporal

### [Projeto 02: Análise de E-commerce Complexa](/02-projeto-intermediario)
**Objetivo:** Análise avançada de e-commerce com múltiplas tabelas relacionadas.

**Principais Descobertas:**
- Diagnóstico crítico: 0% de retenção de clientes (96.478 clientes únicos)
- SP domina 50% do faturamento (R$5M+)
- Nota média de satisfação: 4.09/5 (74.81% satisfação)

**Tecnologias:** MySQL, CTEs, JOINs complexos, Análise de Cohort
